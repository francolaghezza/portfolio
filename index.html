jQuery & jQuery UI

Librería de JavaScript, simplifica la manera de interactuar con el DOM, manejar eventos, desarrollar amnimaciones y aplicaciones AJAX.

Bootstrap

Framework orientado al front-end para realizar aplicaciones web responsive, mobile first, etc. Cuenta con varios componentes y plugins prediseñados.

Symfony

Framework de PHP utilizado para optimizar el desarrollo de aplicaciones web, utiliza MVC, Twig como motor de plantillas y el ORM Doctrine.

Sistemas gestores de bases de datos

Oracle Database

SGBD de tipo objeto-relacional desarrollado por Oracle Corporation. Escrito en C, C++ y Java. Oracle es, básicamente, una herramienta cliente/servidor para la gestión de base de datos.

MySQL

MySQL es un SGBD de datos relacionales de código abierto basado en SQL. Se diseñó y se optimizó para las aplicaciones web y puede utilizarse en cualquier plataforma.

MariaDB

SGDB derivado de MySQL, incorpora las funcionalidades propias de MySQL e incluye otras mejoras, como la incorporación de nuevos motores de almacenamiento.
